{\bfseries{Arcade}} est une plateforme de jeu~\+: un programme qui permet à l\textquotesingle{}utilisateur de choisir un jeu auquel jouer et tient un registre des scores des joueurs.

Avec notre programme, vous pouvez jouer à {\bfseries{Snake}} et {\bfseries{Nibbler}} dans trois styles graphiques différent \+:

\item NCurse
\item Sdl2
\item SFML
